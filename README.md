## Setup
1. In the command line, navigate to project folder
2. Run basic a web server :

  Windows :
  ```
  python -m http.server 8080
  ```

  Linux :
  ```
  python -m SimpleHTTPServer 8080
  ```
3. Visit the URL
[localhost:8080](localhost:8080)
